# Database Logos

This directory is a placeholder for database logos.

## Expected Logos

- `themisdb-logo.svg` - ThemisDB logo
- `postgresql-logo.svg` - PostgreSQL elephant logo
- `mongodb-logo.svg` - MongoDB leaf logo
- `neo4j-logo.svg` - Neo4j logo

## Usage

Logos are referenced in the feature data but not required for the plugin to function.
They can be added later to enhance the visual presentation.

## Format

- SVG format recommended for scalability
- PNG format acceptable (minimum 200x200px)
- Transparent backgrounds preferred
