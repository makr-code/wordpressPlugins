# Changelog

## [1.0.0] - 2026-02-11

### Added
- Initial release
- Interactive feature comparison matrix
- 6 comparison categories (Data Models, AI/ML, Performance, Compatibility, Licensing)
- 40+ features compared across 4 databases (ThemisDB, PostgreSQL, MongoDB, Neo4j)
- Category filtering with interactive buttons
- Column sorting by support level (full > limited > no)
- CSV export functionality
- Mobile card view (responsive design < 768px)
- ThemisDB column highlighting with "⭐ Recommended" badge
- Tooltips for complex features with info icons
- WCAG 2.1 AA accessibility compliance
- Themis brand colors integration (#2c3e50, #3498db, #7c4dff)
- Dark mode support
- Sticky header for easy navigation
- Status icons: ✓ (full), ◐ (limited), ✗ (no)
- Shortcode parameters: category, style, highlight_themis, sticky_header, filterable
- Admin settings page with customization options
- Multiple style variants: modern, compact, minimal
- Keyboard navigation support
- Screen reader compatibility
- High contrast mode support
- Conditional asset loading (only when shortcode present)
- Minified assets for production
- No external dependencies except jQuery
- Performance optimizations with caching

### Features
- **Data Models**: Relational SQL, Graph Database, Document Store, Vector/Embeddings, Time-Series, Key-Value
- **AI/ML Features**: Embedded LLM (ThemisDB exclusive), Vector Similarity Search, RAG Support (ThemisDB exclusive), GPU Acceleration (ThemisDB exclusive)
- **Performance**: Horizontal Scaling, Auto-Sharding, Replication, Built-in Caching
- **Compatibility**: SQL Protocol, MongoDB Protocol, Cypher (Graph), REST API, GraphQL API
- **Licensing**: License Type, Free for Commercial Use, Cloud Vendor Lock-in (inverted scoring)

### Highlights
- ThemisDB has 5+ unique "full" features not available in competing databases
- AI/ML category showcases ThemisDB's superior capabilities
- Embedded LLM, RAG Support, and GPU Acceleration are ThemisDB exclusives
- Interactive filtering makes it easy to compare specific feature categories
- Mobile-responsive design ensures accessibility on all devices
