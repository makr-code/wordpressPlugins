### SEO & Marketing
- **Rank Math SEO** - Schema Markup, XML Sitemap, SEO Analyse
- **Redirection** - 301 Redirects, 404 Monitoring

## Best Practices

### Do's ✅
- Updates wöchentlich durchführen
- Staging-Environment nutzen
- Backup vor Updates
- < 25 Plugins aktiv halten
- Nur notwendige Plugins aktivieren

### Don'ts ❌
- Keine Nulled/Pirated Plugins!
- Inaktive Plugins löschen
- Keine unbekannten Quellen
- Zu viele ähnliche Plugins vermeiden
- Updates niemals ohne Backup

## Troubleshooting

### White Screen of Death
```
1. FTP-Zugriff
2. Alle Plugins deaktivieren
   → wp-content/plugins umbenennen
3. Einzeln wieder aktivieren
4. Schuldiges Plugin finden
```

### Langsame Website
```
1. Query Monitor aktivieren
2. Langsame Queries finden
3. Cache leeren
4. Bilder optimieren
5. Database bereinigen
```

### Security Warning
```
1. Wordfence-Scan starten
2. Malware-Details prüfen
3. Backup zurückspielen
4. Alle Passwörter ändern
5. Sicherheits-Audit
```
