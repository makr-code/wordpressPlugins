Advanced taxonomy management with visual tree UI, TF-IDF analysis, and intelligent term extraction for ThemisDB documentation.

== Description ==

ThemisDB Taxonomy Manager is a powerful WordPress plugin that provides advanced taxonomy management capabilities with a visual tree interface, intelligent term extraction, and TF-IDF (Term Frequency-Inverse Document Frequency) analysis.

**Key Features:**

* Visual tree UI for hierarchical taxonomy management
* Intelligent taxonomy extraction using TF-IDF algorithm
* Custom taxonomies for database features
* Analytics and term importance scoring
* Bulk term operations
* Admin dashboard with visual representations
* Widget support for displaying taxonomies
* Shortcode support for flexible integration
* REST API endpoints for programmatic access
* Schema.org markup for better SEO

**Perfect for:**

* Database documentation websites
* Technical knowledge bases
* Product feature catalogs
* Hierarchical content organization
* SEO-optimized content structure

== Installation ==

1. Upload the `themisdb-taxonomy-manager` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to ThemisDB → Taxonomies to configure your taxonomies

== Frequently Asked Questions ==

= What is TF-IDF analysis? =

TF-IDF (Term Frequency-Inverse Document Frequency) is an algorithm that helps identify the most important terms in your content by analyzing how frequently they appear relative to other documents.

= Can I create custom taxonomies? =

Yes! The plugin supports creating custom taxonomies with full control over labels, slugs, and hierarchical structure.

= Does it work with other plugins? =

Yes! The plugin follows WordPress coding standards and is compatible with most WordPress plugins and themes.

= How do I display taxonomies on my site? =

Use the widget in Appearance → Widgets or use shortcodes: `[themisdb_taxonomy taxonomy="database_features"]` or `[themisdb_taxonomy_info]`

= Can I export/import taxonomies? =

The plugin stores all data in the WordPress database and supports standard WordPress export/import functionality.
