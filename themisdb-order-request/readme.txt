== Description ==

ThemisDB Order Request is a comprehensive order management plugin featuring customer relationship management, contract handling, payment tracking, PDF generation, and license key management.

**Key Features:**

* Complete CRUD operations for orders
* Contract management system
* Payment tracking and invoicing
* PDF generation for contracts and invoices
* License key generation and validation
* Email notifications
* Customer database management
* Training package management
* Support ticket integration
* Order status tracking
* Dashboard with statistics
* RESTful API endpoints
* Admin interface with filters

**Order Management:**

* Create and edit orders
* Track order status (pending, confirmed, completed, cancelled)
* Customer information management
* Product/service selection
* Price calculation
* Tax handling

**Contract Features:**

* Contract generation and management
* Digital contract storage
* Contract status tracking
* Validity period management
* Automatic expiration notifications

**Payment Processing:**

* Payment record tracking
* Invoice generation
* Payment status monitoring
* Multiple payment methods
* Payment history

**License Management:**

* Automatic license key generation
* Core limits and restrictions
* License activation/deactivation
* License validity tracking
* Multi-license support

**Perfect for:**

* Software sales
* Service providers
* Product ordering systems
* License management
* Customer relationship management

== Installation ==

1. Upload the `themisdb-order-request` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to ThemisDB Orders in the admin menu
4. Configure order settings and email templates

== Frequently Asked Questions ==

= Can customers submit orders from the frontend? =

Yes! Use the order request shortcodes to create frontend forms where customers can submit order requests.

= Does it generate PDF documents? =

Yes! The plugin can generate PDF contracts and invoices automatically.

= How are license keys generated? =

License keys are automatically generated using secure random algorithms when an order is confirmed.

= Can I customize email templates? =

Yes! Email templates can be customized through the plugin settings.

= Does it integrate with payment gateways? =

The current version tracks payment records manually. Payment gateway integration would require additional development.

= Can I export order data? =

Order data is stored in the WordPress database and can be exported using standard WordPress export tools.
