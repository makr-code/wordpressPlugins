# ThemisDB Widgets Guide

This guide provides a practical default setup for widget placement with the current ThemisDB theme structure.

## Available Widget Areas

1. Sidebar (`sidebar-1`)
2. Front Page Hero Widgets (`frontpage-hero`)
3. Front Page Content Widgets (`frontpage-content`)
4. Footer Widget 1 (`footer-1`)
5. Footer Widget 2 (`footer-2`)
6. Footer Widget 3 (`footer-3`)

## Recommended Default Layout

### Front Page Hero Widgets (`frontpage-hero`)
Goal: immediate engagement and clear next action.

1. ThemisDB: Call to Action
2. ThemisDB: Newsletter

Tip: Keep this area to 1-2 widgets. More than that weakens the hero focus.

### Front Page Content Widgets (`frontpage-content`)
Goal: deepen interest before users continue to latest posts.

1. ThemisDB: Popular Posts
2. ThemisDB: Quick Links

Alternative set for community-driven sites:
1. ThemisDB: Testimonials Carousel
2. ThemisDB: Social Links

### Sidebar (`sidebar-1`)
Goal: support navigation and content discovery.

1. Search (core widget/block)
2. ThemisDB: Tabbed Posts
3. ThemisDB: Category Grid
4. ThemisDB: Tag Cloud

### Footer Widgets (`footer-1` to `footer-3`)
Goal: utility and trust signals.

1. Footer 1: About Box
2. Footer 2: Quick Links
3. Footer 3: Social Links

## Minimal High-Conversion Preset

If you want a lean setup that is easy to maintain:

1. `frontpage-hero`: CTA Widget
2. `frontpage-content`: Popular Posts Widget
3. `sidebar-1`: Search + Tabbed Posts
4. `footer-1`: About Box
5. `footer-2`: Quick Links
6. `footer-3`: Social Links

## Setup Steps (WordPress Admin)

1. Go to Appearance > Widgets.
2. Open `Front Page Hero Widgets` and add your CTA widget.
3. Open `Front Page Content Widgets` and add Popular Posts and/or Quick Links.
4. Fill `Sidebar` and the 3 footer areas based on the preset above.
5. Open the static front page and verify spacing and hierarchy on desktop and mobile.

## Practical Rules

1. Do not place multiple moving widgets in the same area (for example slider + ticker + carousel).
2. Prefer one primary action per area.
3. Keep text widgets short in hero-adjacent zones.
4. Re-check after major content changes: widgets can become redundant when homepage sections already cover the same information.
