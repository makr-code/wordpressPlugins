Technische Details

Die Graph-Navigation verwendet nun eine PHP-Funktion (themisdb_get_graph_data()), die WordPress-Inhalte abruft und an JavaScript übergibt. Die Daten werden per wp_localize_script() als themisdbGraphData bereitgestellt.

Das JavaScript überprüft zuerst, ob WordPress-Daten verfügbar sind, und erstellt daraus eine interaktive Force-directed Graph Visualisierung. Falls keine Daten vorhanden sind, wird auf die menübasierte Darstellung zurückgegriffen.

⚠️ Hinweis für Testen

Um die Graph-Navigation vollständig zu testen, muss das Theme in einer echten WordPress-Installation aktiviert sein. In dieser Demo-Umgebung werden Mock-Daten verwendet, um die Funktionalität zu demonstrieren.
