FAQ

Ist ThemisDB kostenlos? +

Ja. ThemisDB Core ist Open Source (Apache 2.0) und vollständig kostenlos. Für Enterprise-Features wie Premium-Support, erweiterte HA-Tooling und SSO gibt es kostenpflichtige Pläne. Alle Details unter /pricing.

Kann ich von PostgreSQL zu ThemisDB migrieren? +

ThemisDB ist weitgehend PostgreSQL-protokollkompatibel. Mit dem enthaltenen Migrationstool themis-migrate können Sie pg_dump-Exporte direkt importieren. Mehr dazu: Migrations-Guide.

Welche Betriebssysteme werden unterstützt? +

Linux (x86_64, ARM64), Windows 10/11 (64-bit), macOS 12+ (Intel und Apple Silicon) sowie Docker. Vollständige Liste unter /download.

Wie bekomme ich Sicherheitsupdates schnell mit? +

Abonnieren Sie die Sicherheits-Mailingliste oder aktivieren Sie GitHub-Notifications für das Repository. Enterprise-Kunden erhalten direkte E-Mail-Benachrichtigungen.

Wo melde ich Sicherheitslücken? +

Sicherheitslücken bitte nicht öffentlich als GitHub-Issue melden. Senden Sie einen verschlüsselten Bericht an security@themisdb.org. Unser PGP-Key ist auf der Sicherheitsseite verfügbar.

Gibt es einen Kubernetes-Operator? +

Ja. Der ThemisDB Kubernetes Operator ist auf GitHub verfügbar und unterstützt automatischen Scale-up/down, Rolling Upgrades und integriertes Backup via Velero. Doku: /dokumentation/kubernetes.
